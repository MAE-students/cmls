\begin{abstract}
    The main topic of this homework is timbre transfer. The goal is to implement an algorithm that is able to transfer the timbre of a particular source to a target audio signal. In order to solve such a task, it is necessary to rely on signal processing methods, in particular on Nonnegative Matrix Factorization (NMF). The implementation is carried out in a Jupyter Notebook file named \textit{"DeToni\_Vecchini.ipynb"}. This file contains the code which is explained step by step in this report. Additionally, the Jupyter Notebook will be a reference while reading the report, as certain figures and parts of the code are mentioned when relevant.
\end{abstract}
